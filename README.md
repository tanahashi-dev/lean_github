# lean_github

## 📋 プロジェクト概要

シンプルな Python スクリプトと unittest テストで構成されたプロジェクトです。
現在の日時を表示する挨拶スクリプトを実装し、テスト駆動開発の考え方を取り入れています。

## 🎯 機能一覧

- ✅ 現在の日時を表示する挨拶スクリプト
- ✅ 自動テスト機能付き
- ✅ 時刻フォーマット機能
- ✅ モジュール化されたコード構造

## 📂 ファイル構成

| ファイル | 説明 |
|----------|---------|
| `hello.py` | メインの挨拶スクリプト |
| `test_hello.py` | unittest によるテストケース |
| `README.md` | 本文書 |

## 🚀 使用方法

### スクリプトを実行

```bash
python hello.py
```

### テストを実行

```bash
python -m unittest test_hello.py
```
